Linked separate products together by product attributes and More. Needs WooCommerce to work.

== Description ==
Linked products by selected woocommerce attributes, a simple way to handle complex product variation.

If you wish to take a look at this plugin how it works then [see demo](https://demos.devsace.com/plugins/linked-variation-for-woocommerce/product/iphone-12-black-64gb).

= Core Features =
* Create swap links between separate products in your store.
* Link different versions of a product together without using variations.
* Shorting variations and able to on/off product image.

To know how to use, visit the website for [video documentation](https://demos.devsace.com/plugins/linked-variation-for-woocommerce/video-documentation/).

== Installation ==
1. It can be installed directly through your WordPress Plugins dashboard.
1. Click "Add New" and search for "Linked Variation for WooCommerce"
1. Install and Activate
1. Go to Product > Woo Linked Variations and "Add New"
1. Select multiple products to linked together, that's all

Alternatively, you can download the plugin using the download button on this page and then upload the woocommerce-linked-variationsÂ folder to the /wp-content/plugins/ directory then activate through the Plugins dashboard in WordPress
